# Vouchins Brand Guidelines

- **Button Styling Rule**: ALL buttons in the Vouchins application MUST strictly adhere to the brand colors. Do not use generic colors (like unbranded blue, red, or green) for buttons like "Get Early Access", "Join Now", etc. The primary brand color for dark backgrounds is `#0A1B5C` and `#4FD1C5` for accents. Primary buttons should typically use the primary tailwind variable or specific brand hex codes to ensure consistency.
- **Typography Rule**: Never use a double hyphen (`--`) or em-dash (`—`) in copy. Always replace them with a single hyphen (`-`) surrounded by spaces where applicable.
